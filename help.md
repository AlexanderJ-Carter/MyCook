# 使用帮助

欢迎来到 MyCook！本页将帮助你快速上手。

## 基本使用

### 浏览菜谱

1. **首页导航**：访问首页可以看到两大菜谱来源的入口卡片
2. **侧边栏**：点击左上角的菜单图标展开侧边栏，按分类浏览
3. **搜索功能**：按 `/` 键或点击顶部搜索框，输入菜名或食材搜索

### 搜索技巧

- 支持**模糊匹配**：输入"西红柿"可以找到所有包含西红柿的菜谱
- 支持**拼音搜索**：输入"xhsc"可以匹配"西红柿炒蛋"
- 搜索结果按相关度排序

### 阅读体验

- **阅读进度条**：页面顶部显示阅读进度
- **返回顶部**：右下角按钮可快速返回顶部，显示当前进度
- **目录导航**：右侧目录栏可快速跳转到各个章节
- **上一道/下一道**：页面底部可切换相邻菜谱

### 深色模式

- 点击顶部导航栏的太阳/月亮图标切换
- 自动跟随系统设置

## 内容说明

### CookLikeHOC

- 来源：《老乡鸡菜品溯源报告》整理
- 分类方式：按烹饪方式（炒菜、炖菜、卤菜、汤等）
- 特点：详细的步骤说明，适合学习基础烹饪技巧

### HowToCook

- 来源：程序员在家做饭方法指南
- 分类方式：按食材类型（素菜、荤菜、水产等）
- 特点：程序员风格，简洁明了，适合快速上手

### HowToCook 图片版

- 位置：`/howtocook-images/`
- 说明：4K 高清菜谱图，适合收藏或打印

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `/` | 聚焦搜索框 |
| `Esc` | 关闭搜索 |
| `←` / `→` | 上/下一页（部分页面支持） |

## 移动端使用

- 侧边栏通过左上角菜单图标访问
- 搜索框在顶部导航栏
- 卡片布局自动调整为单列

## 打印菜谱

1. 打开想要打印的菜谱页面
2. 按 `Ctrl+P`（Mac: `Cmd+P`）
3. 导航栏和侧边栏会自动隐藏
4. 选择打印机或保存为 PDF

## 常见问题

### 搜索找不到某道菜？

- 检查菜名是否正确
- 尝试搜索食材名称
- 该菜可能尚未收录

### 页面显示异常？

- 尝试刷新页面（F5）
- 清除浏览器缓存
- 使用最新版本的浏览器

### 想要贡献菜谱？

菜谱内容来自上游项目，请到对应仓库贡献：

- [HowToCook](https://github.com/Anduin2017/HowToCook)
- [CookLikeHOC](https://github.com/Gar-b-age/CookLikeHOC)
